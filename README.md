# Musteri-Genel-Analiz
Ticaret yapan bir işletmenin verilerinden yola çıkarak bu raporu oluşturdum. Bu rapor ile müşteri performansını takip ederek satış ekibini analitik olarak destekledim. Müşterinin geçen seneye göre performansını kümülatif toplamlarla ölçtüm. Satışları geçen seneye göre kümülatif karşılaştırarak müşterinin satış performans analizlerini oluşturdum.
